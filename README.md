# Carte de Noël Interactive

## Description
Une carte de Noël interactive avec des animations de neige, de la musique et un effet de swipe.

## Fonctionnalités
- Animation de neige dynamique
- Swipe de carte
- Musique de Noël
- Design responsive

## Technologies
- HTML5
- CSS3
- JavaScript

## Déploiement
Hébergé sur Vercel
